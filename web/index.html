Glider Project

A 3D graphics application simulating a flying glider.

Download:

Windows 10 x64: Glider Project

Instructions:

Download and unzip the .zip file.
Open the Glider Project folder. Open the glider_project app.

Licence:

Glider Project is free and open source. You can find the source code here.

Description:

Glider Project is a 3D simulation of a flying hang glider.
User can control the glider's movement.
